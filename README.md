# LeadNotifyBot Landing

Лендинг для SaaS-проекта LeadNotifyBot — Telegram CRM для заявок малого бизнеса.

## Как запустить

```bash
npm install
npm run dev
```

## Главные файлы

- `src/App.jsx` — сборка страницы
- `src/styles.css` — все стили
- `src/data/siteData.js` — тексты карточек, ниш и FAQ
- `src/components/` — блоки лендинга
